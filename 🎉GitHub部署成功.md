1. 在 Vercel Dashboard 中点击 **"New Project"**
2. 在项目列表中找到 `TianxiangDU/cointoon-demo`
3. 点击 **"Import"**

**第三步：配置部署**
保持默认配置即可：
```

Build Command: npm run build
Output Directory: .next

```

1. 点击 **"Deploy"**
2. 等待 2-3 分钟完成构建和部署
3. 部署成功后获得访问链接
