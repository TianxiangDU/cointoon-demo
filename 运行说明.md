# CoinToon Demo 运行说明

## 🎉 项目已成功创建！

您的 CoinToon Demo 站点已经完成搭建，包含以下功能：

### ✅ 已完成的功能

1. **主页 (http://localhost:3000)**
   - CTN 主币信息展示
   - 生态系统统计数据
   - 热门角色代币网格展示
   - 参与流程说明

2. **角色详情页 (/roles/[id])**

### 📁 项目结构

```
cointoon-demo/
├── src/
│   ├── app/
│   │   ├── page.tsx              # 首页
│   │   ├── creators/page.tsx     # 创作者页面
│   │   ├── roles/[id]/page.tsx   # 角色详情页
│   │   └── globals.css           # 全局样式
│   └── components/
│       ├── Layout.tsx            # 页面布局
│       └── RoleCard.tsx          # 角色卡片组件
├── data/
│   ├── roles.json               # 角色数据
│   └── ctn-info.json           # CTN 主币信息
└── public/images/               # 图片资源
```

## 🚀 如何运行

### 1. 启动开发服务器
```bash
cd cointoon-demo
npm run dev
```

### 2. 访问网站
打开浏览器访问：http://localhost:3000

### 3. 构建生产版本
```bash
npm run build
npm start
```

## 🔧 自定义配置

### 添加新角色
1. 编辑 `data/roles.json` 添加角色数据
2. 将角色图片放入 `public/images/` 目录
3. 重启开发服务器

### 修改主币信息
编辑 `data/ctn-info.json` 文件更新 CTN 主币信息

### 样式自定义
- 全局样式：`src/app/globals.css`
- 组件样式：使用 Tailwind CSS 类名
- 配置文件：`tailwind.config.js`
